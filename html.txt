em(i) - тег для курсива
b - жирный текст 
<strike> - перечеркнутый текст
<sub> - для нижнегго регистра цифр
<sup> - для верхнего регистра цифр
<hr> - горизонтальная линия на всю ширину страницы
<abbr> - при ипользовании атрибута "title" - при наведении мышки расшифровывается текст
<address> - для добавления адреса на страницу

<small>, <big> - чуть меньше, чуть больше текст
<pre> - для сохранения всех пробелов, пропусков строк
<code> - для записи кода на странице

<ol>(li) - атрибуты start"1", type"a".
<ul> - атрибуты type="circle" - круглые(полые), "square" - квадратные(закрашенные)

                       Таблицы в HTML
<table> - обвертка таблицы
атрибут border="1", width="500" 
<thead> - верхний колонтитул таблицы
<tbody> - основная часть таблицы
атрибут align="center"
<tfoot> - нижний колонтитул таблицы
<tr> - строка (ряда) таблицы
<td> - обычная ячейка таблицы
<th> - ячейка-заголовка таблицы

<col> - колонка таблицы
<colgroup> - группы колонок таблицы 
<caption> - подпись таблицы


script - в конце html body


<div> - блочный для создания  блоков
<span> - строчный для придания css свойст нужному слову


<form> - для создания форм 
   action - указать ту страницу для обработки формы
   method - метод передачи данных (get, post)
<label> - создать подпись для поля ввода
   for="username" - при нажатии на слово становится активным поле

<input> - поле ввода
   placeholder - подсказка отображается в поле текста
   value - сначала стереть потом вводить
   maxlength="5" - максимальное количество символов которые можно ввести в поле
   hidden - скрыть поле
   disabled - не активное поле 
   readonly
   name - выделить разные input поля через javascript
   id="" - установить к любому тегу(уникальный идентификатор) - возможность через css к id и добавлять стили

<textarea></textarea> - комментарии к сайту
   required - не отправиться на сервер если ничего не будет введено

<button></button> - использовать кнопку вне формы
   type="reset, button"


Селекторы выбора 

<select></select> - выпадающий список 
   multiple - выбрать несколько в списке
   size - для указания сколько будет показано варинтов выбора 
  <option></option> - для ввода выпадающих списков
   selected - для выбора по умолчанию

 <summary> - для ввода текста
